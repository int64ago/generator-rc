# generator-rc
[![NPM version][npm-image]][npm-url] [![Dependency Status][daviddm-image]][daviddm-url]
> React component generator

## Usage

```bash
npm install -g yo
npm install -g @hspkg/generator-rc
```

Then in your project folder:

```bash
yo @hspkg/rc
```
